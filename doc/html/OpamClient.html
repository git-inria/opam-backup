Client entry-point.
val init : OpamTypes.repository -> OpamTypes.compiler option -> int -> unit
Initialize the client a consistent state. init repo alias oversion cores means:
  • repo is the initial repository description,
  • oversion is the version of the compiler. None means we are using the system-wide installed compiler.
  • cores is the number of cores

val upgrade : OpamPackage.Name.Set.t -> unit
Find a consistent state where most of the installed packages are upgraded to their latest version. If no package packages are specified then try to upgrade everything. Take the global file lock.
val upload : OpamTypes.upload -> OpamTypes.repository_name option -> unit
Upload a package to a remote repository. If repo is None then it will look for the repository associated with the package name. Otherwise, it will look for a repository having the right name. Take the global file lock.
