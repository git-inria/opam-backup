OPAM client.

Initializes the client a consistent state.
val list : unit -> unit
Displays all available packages
val info : Types.N.t -> unit
Displays a general summary of a package.
val config : Types.config -> unit
Depending on request, returns options or directories where the package is installed.
val install : Types.N.t -> unit
Installs the given package.
val update : unit -> unit
Refresh the available packages.
val upgrade : unit -> unit
Finds a consistent state where most of the installed packages are upgraded to their latest version.
val upload : Types.upload -> string option -> unit
Upload a package to a remote repository. If repo is None then it will look for the repository associated with the package name. Otherwise, it will look for a repository having the right name.

Removes the given package.

Manage remote repositories
